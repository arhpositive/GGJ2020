This is where you create levels.
naming convention: levelxx.csv (xx has to be the number in which the level will appear)

First line
camera size; camera posx; camera posy; cameraposz; movelimit;

Rest of document
tile;tile;tile;tile;.....
tile;tile;tile;tile;.....

tile codes
0: empty (removed) tile
1: void tile
2: start tile
3: goal tile
